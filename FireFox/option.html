本ツールについて
  • 本ツールは非公式拡張機能です。X公式とは関係ありません。
  • ツールの使用は自己責任でお願いします。
  • Androidブラウザでも使用できます（Chrome拡張機能に対応しているブラウザが必要ですが）。ブラウザの機能でホーム画面にXを配置するとブラウザUIが非表示になりアプリのように使えます（PWA）
  • 不具合などある場合は以下にレビューまたはissuesで報告いただければ可能な範囲で対応したいと思います
    • Firefox Browser ADD-ONS
    • GitHub
